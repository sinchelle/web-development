##这是一个关于百分比布局图片轮播

###注：通常情况下，不会使用到这种方式

##整体思路

最近在做一个带有背景图来约束板心的项目，所以轮播图整体也是用的百分比写的宽高，这就涉及到，当用户缩放时，浏览器窗口的宽高发生了变化，而所写的css样式里的宽高只在页面加载的时候，获取了一次，并没有随着浏览器窗口大小的改变而改变，所以在轮播图里的js中，写了一个监听浏览器窗口改变的函数，方法为window.onresize(function(){});jquery为$(window).resize(function(){});当我手动调整浏览器宽度时，清除定时器，让图片停止轮播，并获取轮播图所在盒子的宽度后重新开启定时器。
